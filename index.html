Image Search Abstraction Layer

Project by Soirana

User stories:
  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset={number} parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example usage:

https://soirana-image-search.herokuapp.com/grumpycats
https://soirana-image-search.herokuapp.com/grumpycats?offset=2
https://soirana-image-search.herokuapp.com/latest/search

Example output:

{"url": "https://api.imgur.com/3/gallery/search/{search term}", "title": "yoga and cats"}
{"error": "page seems to be empty"}
{"search":"dogs","page":1,"when":"14/5/2016 - 12:40:52"}
